# 問題:Can you execute ?
## 概要
ファイルの種類を調べて実行する．



## 解き方
fileコマンドを使用し，ファイルの種類を確かめる．コマンドの実行方法は以下の通り．

``` 
file exec_me
```

実行結果を見ると次のような情報が得られる．

- ファイル形式: ELF 64-bit LSB executable
- アーキテクチャ: x86-64
- 動的リンク
- インタープリター: /lib64/ld-linux-x86-64.so.2
- 対応OS: GNU/Linux 2.6.24 以降

得られた情報をもとにLinux環境で実行する．

実行方法は以下の通り．

``` 
./exec_me
```

実行するとFlagが得られる．
